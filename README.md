# Spendly – Personal Expense Tracker 💸

Spendly is a modern, lightweight web application designed to help users track their daily expenses effortlessly. With a focus on privacy and speed, all data is stored locally in your browser.

## ✨ Features
- **Modern Dark UI:** Clean, premium interface using the Sora and Playfair Display typefaces.
- **Dynamic Dashboard:** Real-time calculation of remaining budget, daily averages, and top spending categories.
- **Visual Analytics:** Interactive bar charts showing spending trends over the last 14 days (powered by Chart.js).
- **Flexible Management:** Add, delete, and filter transactions by category.
- **Data Portability:** Export your financial data to CSV or send reports via email.
- **Privacy First:** No database required; everything stays on your device via LocalStorage.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript.
- **Charts:** [Chart.js](https://www.chartjs.org/)
- **Fonts:** Google Fonts (Sora & Playfair Display)
